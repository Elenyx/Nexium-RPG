Bot Commands

Complete list of available slash commands and their usage

‚öîÔ∏è Core Commands

/help

Core

Get help and information about the bot, including getting started guide and available features.

Usage: /help
Interactive Buttons Feature Overview Support Links

/profile

Core

View your dimensional profile or another user's profile, including stats, inventory, and collection.

Usage: /profile [user]
Options:
  • user - Optional user to view profile of
Interactive UI Stats Display Collection View

/ping

Core

Check the bot's latency and response time to ensure optimal performance.

Usage: /ping
Bot Latency API Latency Status Check

/pull

Core

Pull for random characters using coins. Higher rarity characters have lower drop rates.

Usage: /pull [amount]
Options:
  • amount - Number of pulls (1-10, costs 100 coins each)
Gacha System Character Summoning Rarity Drops

/collection

Core

View your character collection as a beautiful card album with pagination.

Usage: /collection [page] [user]
Options:
  • page - Page number to view
  • user - Optional user to view collection of
Card Album Pagination Visual Display

/battle

Core

Battle between two of your characters to test their abilities and strategies.

Usage: /battle character1 character2
Options:
  • character1 - ID of first character
  • character2 - ID of second character
Turn-based Combat Ability System Strategic Gameplay

/upgrade

Core

Upgrade your characters using shards. Supports both rarity upgrades and level progression.

Usage: /upgrade character_id [type]
Options:
  • character_id - ID of the character to upgrade
  • type - Type of upgrade (rarity/level)
Rarity Upgrades Level Progression Two-Group System

/rarity-progress

Core

Check the rarity upgrade progress for a specific character, including required shards.

Usage: /rarity-progress character_id
Options:
  • character_id - ID of the character to check
Progress Tracking Shard Requirements Upgrade Path

/quest

Core

Complete quests to earn shards, coins, and other rewards. Includes daily and achievement quests.

Usage: /quest [type]
Options:
  • type - Quest type (main_story, daily, side_quest, achievement)
Daily Quests Reward System Achievement Tracking
